# 実装計画: ミステリアスなGateWay画面の実装

## 1. 概要
「The Silent Order」の入り口となるGateway画面を実装する。ユーザーがアクセスした瞬間に、秘密結社の怪しくも高貴な雰囲気に没入できるリッチなUIを構築する。

## 2. 変更内容の設計

### 2.1 依存関係の追加
- `framer-motion`: アニメーション（浮遊感、タイピング、フェードイン）
- `lucide-react`: アイコン（秘密結社風のシンボル）

### 2.2 スタイリング方針
- `globals.css` の更新:
    - 背景色: `zinc-950` (#09090b)
    - 強調色: `amber-500` (#f59e0b) - 黄金の輝き
    - 恐怖色: `rose-900` (#881337) - 深紅のアクセント
    - 煙のようなグラデーション効果の定義

### 2.3 コンポーネント設計
1. **MysticBackground.tsx**
    - `framer-motion` を使用した浮遊するパーティクルまたは煙のエフェクト。
    - 背景に深いグラデーションを配置。
2. **CrypticText.tsx**
    - タイピングエフェクト。
    - メッセージ: 「警告：これより先、汝の知識が試される。覚悟なき者は立ち去れ。」
    - わずかなグリッチエフェクトを追加。
3. **GatewayButton.tsx**
    - 「試練を開始する」ボタン。
    - 黄金のボーダーと、ホバー時に広がる光彩（Glow）演出。

### 2.4 ページ構成 (src/app/page.tsx)
- 全画面を `MysticBackground` で覆う。
- 中央に `CrypticText` を配置し、完了後に `GatewayButton` をフェードイン。

## 3. 影響範囲
- `src/app/page.tsx`: 既存のデフォルトコンテンツを完全に置き換え。
- `src/app/globals.css`: グローバルな背景色とカラーパレットの更新。

## 4. 検証計画
- **表示確認**: 漆黒の背景に黄金のテキストが美しく映えているか。
- **アニメーション**: タイピングエフェクトとボタンのフェードインがスムーズか。
- **レスポンシブ**: スマートフォンでも雰囲気が損なわれていないか。
